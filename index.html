NNACC

Acceleration Framework Using NN Method

We designed this tool with intention to realize a functional interface for using NN algorithms: Just provide a set of input-output data pairs as the behavior description of a module. Then the framework will automatically pick up an Neural Network configuration to fulfill your demand. With support from several backend(CPU, GPU, etc.), this framework can execute with satisfying speed.

Present this framework is rudimentary, we hope if more people hold the same perspective or just interested in this project and join us and make the contribution:)
